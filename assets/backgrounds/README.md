# HarborVane SVG backgrounds

Original full-bleed / hero-scale SVGs using colors from `brand/BRAND.md`.

| File | Vibe |
|------|------|
| `hv-mesh-bloom.svg` | Soft mesh orbs, navy → teal (hero) |
| `hv-wave-layers.svg` | Horizontal wave bands, maritime |
| `hv-hex-tide.svg` | Subtle hex grid + teal strokes |
| `hv-gold-drift.svg` | Gold particles / drift lines on navy |
| `hv-ripple-berth.svg` | Concentric ripples, calm dock water |

**Usage:** embed as `<img>`, CSS `background-image: url(...)`, or inline. For Next/Hugo, place in `static/` and reference by path.
