# HarborVane SVG backgrounds

Original hero-scale SVGs aligned with **[`brand/BRAND.md`](../brand/BRAND.md)** (canonical: [`harborvane/harborvane` `BRAND.md`](https://github.com/harborvane/harborvane/blob/main/BRAND.md)).

**Palette:** Primary Navy `#0B2A45`, Teal `#1F6F7A`, Gold `#F2C14E`, Navy Light `#1a2332`, White `#FFFFFF`.

| File | Vibe |
|------|------|
| `hv-mesh-bloom.svg` | Soft mesh orbs, navy → teal (matches primary background gradient spirit) |
| `hv-wave-layers.svg` | Horizontal wave bands, maritime |
| `hv-hex-tide.svg` | Subtle hex grid + teal strokes |
| `hv-gold-drift.svg` | Gold guidance specks on navy |
| `hv-ripple-berth.svg` | Concentric ripples, calm water |

**Usage:** `<img>`, CSS `background-image`, or inline SVG. Not a substitute for official logo files in `BRAND.md`.

**Note:** Pattern *style* is inspired by layered SVG backgrounds (e.g. [SVGBackgrounds.com](https://www.svgbackgrounds.com/)); these paths are original and use Harborvane colors only.
