# HarborVane — brand tokens (from harborvane.io)

Pulled from production **`https://harborvane.io/styles.css`** (2026-03-19). If the site diverges, refresh from CSS.

## Colors

| Token | Hex | Use |
|-------|-----|-----|
| **Navy deep** | `#1f3a57` | Primary dark surfaces, hero depth |
| **Navy mid** | `#2a4a6f` | Panels, cards, hover states |
| **Teal accent** | `#3fa9a6` | CTAs, links, highlights |
| **Teal tint** | `#3fa9a633` / `#3fa9a61a` | Glass, borders |
| **Gold accent** | `#d4af37` | Premium touches, anchors, dividers |
| **Gold bright** | `#e0c456` | Hover gold, subtle shine |
| **Gold wash** | `#d4af371a` / `#d4af3780` | Background glow |
| **Ink** | `#374151` | Body text on light |
| **Mist** | `#f3f4f6` | Light section backgrounds |
| **White** | `#ffffff` | Text on dark, cards |

## Motion / texture (for SVG)

- Prefer **slow radial gradients** and **low-contrast layers** so text stays readable.
- **Navy → teal** transitions read as “water / systems depth.”
- **Gold** sparingly: wayfinding, not noise.
